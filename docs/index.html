layout

layout: Uint8Array[] = ...
  • Defined in src/core/layout.ts:14

A binary pattern that describes the standard octant layout:

   3____7
 2/___6/|
 | 1__|_5
 0/___4/

This common layout is crucial for positional assumptions.
